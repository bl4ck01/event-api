# event-api
Flask Event Management Rest Api

### Installation

```
virtualenv venv

pip install -r requirements.txt

Rename secret.py.template inside app directory to secret.py and edit configurations
```

### run
```
bash build.sh <env>
<env: prod | dev>
```
